## インストール

1. リポジトリをクローン

   ```sh
   git clone https://github.com/yourusername/textlint-sample-project.git

   ```

2. 依存関係をインストール

   ```
   $ npm install
   ```

3. VS Codeの[vscode-textlint](https://marketplace.visualstudio.com/items?itemName=taichi.vscode-textlint)という拡張機能をインストール
